# Gnosis Mobile - Your Spiritual Companion

A comprehensive Catholic spiritual companion mobile application built with Flutter. Features Bible reading, prayers, rosary guide, AI spiritual chat, devotional music, community groups, and saints library.

## Features

- **Bible Reader**: Complete Bible with bookmarks, highlights, and search
- **Prayers Library**: Traditional Catholic prayers with categories and favorites
- **Holy Rosary**: Interactive guided rosary with all four mysteries
- **AI Spiritual Chat**: AI-powered spiritual guidance with biblical wisdom
- **Devotional Music**: Music player with offline playback and playlists
- **Community Groups**: Real-time chat and fellowship groups
- **Saints Library**: Biographies and stories of Catholic saints
- **Saved Verses**: Bookmark and organize favorite Bible verses
- **Offline Support**: Read prayers, Bible, and listen to music offline
- **Dark/Light Theme**: Beautiful sacred theme with dark mode support

## Prerequisites

- Flutter SDK 3.16.0 or higher
- Dart SDK 3.2.0 or higher
- Android Studio or VS Code with Flutter extension
- iOS: macOS with Xcode (for iOS development)
